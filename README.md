# 基于Vue2.0的典型电商案例

> [在线访问](nickj.leanapp.cn)，目前仅完成部分前端工作



## Build Setup

``` bash
# clone project
cd web-project

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
前端:

- Vue2.0
- Vue-Router
- Vue-Resource
- Vuex
- Vux

后端:

- Node.js

- LeanCloud
